# Car Price Prediction with Machine Learning

## Oasis Infobyte Data Science Internship

**Task 3 – Car Price Prediction with Machine Learning**

### Objective

Build a machine learning regression model that predicts the selling price of a used car based on features such as brand, age, mileage, fuel type, and transmission.

### Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook
